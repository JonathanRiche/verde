What this prototype proves

  • The bundle can be built as a single browser-ready JS asset.
  • The overlay ignores its own DOM so hover targeting remains stable.
  • The inspector emits structured payloads for later Zig and CEF bridging.
  • The fixed prompt UI can coexist with page content at a top-level z-index.
